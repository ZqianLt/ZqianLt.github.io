Function limit

2024/10/23
Note

1.Squeeze Theorem.

  如果函数$f(x),g(x)$及$h(x)$满足以下条件：

$(1).$当$x\in \mathring{U}\left( x_0,r \right) ($或$|x| > M )$时，$g\left( x \right) \leqslant f\left( x \right) \leqslant h\left( x \right) ;$
$(2).\underset{x\rightarrow x_0,\left( x\rightarrow \infty \right)}{\lim g\left( x \right) =A} , \underset{x\rightarrow x_0,\left( x\rightarrow \infty \right)}{\lim h\left( x \right) =A}$ 那么$\underset{x\rightarrow x_0,\left( x\rightarrow \infty \right)}{\lim h\left( x \right) }$存在且等于$A$.


$\mathrm{Eg}1.$ $(2004). \underset{n\rightarrow \infty}{\lim}\sqrt[n]{\sin 1+\sin \frac{1}{2}+\cdots +\sin \frac{1}{n}} $


解
$ \because \underset{n\rightarrow \infty}{\lim}\sqrt[n]{\sin 1}=\underset{n\rightarrow \infty}{\lim}\left( \sin 1 \right) ^{\frac{1}{n}}=1 $ 以及 $ \underset{n\rightarrow \infty}{\lim}\sqrt[n]{1+\frac{1}{2}+\cdots +\frac{1}{n}}=1 $

$ \therefore $原式$=1$

$\mathrm{Eg}2.$ $ \underset{n\rightarrow \infty}{\lim}\left( \frac{n}{n+\pi}+\frac{n}{n+2\pi}+\cdots +\frac{n}{n+n\pi} \right) $


解
$1$

$\mathrm{Eg}3.$ $(2013). \underset{n\rightarrow \infty}{\lim}\left( \frac{1}{n^2+n+1}+\frac{2}{n^2+n+2}+\cdots +\frac{n}{n^2+n+n} \right) $


解
$1/2$

$\mathrm{Eg}4.$ $ \underset{x\rightarrow 0^+}{\lim}x\ln x $


解
$ 0\leqslant \underset{x\rightarrow 0^+}{\lim}|x\ln x|=\underset{x\rightarrow 0^+}{\lim}|\frac{2\ln \sqrt{x}}{\frac{1}{x}}|=\underset{x\rightarrow 0^+}{\lim}|-2\frac{\ln \frac{1}{\sqrt{x}}}{\frac{1}{x}}|\leqslant \underset{x\rightarrow 0^+}{\lim}|-2\frac{\ln \frac{1}{\sqrt{x}}}{\frac{1}{x}}|=\underset{x\rightarrow 0^+}{\lim}|2\sqrt{x}|=0 $

$\mathrm{Eg}5.$ $(2008).\underset{n\rightarrow \infty}{\lim}\sum_{k=1}^n{\frac{k}{n^2+k}}$


解
$1/2$

$\mathrm{Eg}6.$ $(2017).\underset{n\rightarrow \infty}{\lim}\left( \frac{1}{n^4+n^3+n^2+1}+\frac{2}{n^4+n^3+n^2+2}+\cdots \frac{n^2}{n^4+n^3+n^2+n^2} \right) $


解
$1/2$


2.两个重要极限.

  $(1).\underset{x\rightarrow 0}{\lim}\frac{\sin x}{x}=1$

  $(2).\underset{x\rightarrow 0}{\lim}\left( 1+x \right) ^{\frac{1}{x}}=e$


$\mathrm{Eg}1.$ $ \underset{x\rightarrow \infty}{\lim}\frac{3x^2+5}{5x+3}\sin \frac{2}{x} $


解
$6/5$

$\mathrm{Eg}2.$ $ \underset{x\rightarrow 0}{\lim}\frac{3\sin x+x^2\cos \frac{1}{x}}{\left( 1+\cos x \right) \ln \left( 1+x \right)} $


解
$3/2$

$\mathrm{Eg}3.$ $ \underset{x\rightarrow 0}{\lim}\left( 1+x \right) ^{\small{\frac{2}{x}}} $


解
$e^2$

$\mathrm{Eg}4.$ $ \underset{x\rightarrow \infty}{\lim}\left( \frac{x+2a}{x-a} \right) =8 $ ,则$a=(~~~).$


解
$\ln 2$

$\mathrm{Eg}5.$ $ \underset{x\rightarrow 0}{\lim}\left( \sin \frac{1}{x}+\cos \frac{1}{x} \right) ^x $


解
$e$

$\mathrm{Eg}6.$ $ \underset{x\rightarrow +\infty}{\lim}\left[ \sqrt{4x^2+x}\ln \left( 2+\frac{1}{x} \right) -2x\ln 2 \right] $


解
$ \frac{1}{4}\ln 2+1 $

$\mathrm{Eg}7.$ $ \underset{x\rightarrow 0}{\lim}\frac{\left( 1+x \right) ^{\frac{1}{x}}-\left( 1+2x \right) ^{\frac{1}{2x}}}{x} $


解
$e/2$

$\mathrm{Eg}8.$ $(2020).\underset{x\rightarrow \frac{\pi}{2}}{\lim}\left( \sin x \right) ^{\frac{1}{1-\sin x}}$


解
$e^{-1}$

$\mathrm{Eg}9.$ $(2006).\underset{x\rightarrow \infty}{\lim}\left( \frac{x+1}{x-1} \right) ^x $


解
$e^{2}$
