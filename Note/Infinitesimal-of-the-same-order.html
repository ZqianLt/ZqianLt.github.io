Infinitesimal of the same order

2024/10/24
Note

1.Infinitesimal of the same order.

$(1).x\rightarrow 0 $时，$ \mathrm{arc}\tan x\sim \sin x\sim x\sim \mathrm{arc}\sin x\sim \tan x .$
这一组中任意两个函数之差都是，$x^3$ 的同阶无穷小

$(2).x\rightarrow 0 $时，$ \mathrm{arc}\tan x\sim \sin x\sim x\sim \mathrm{arc}\sin x\sim \tan x .$
这一组中任意两个函数之差都是，$x^2$ 的同阶无穷小

$(3). \log _a\left( 1+x \right) \sim \frac{x}{\ln a}, a^x-1\sim x\ln a, \ln \left( x+\sqrt{1+x^2} \right) \sim x, \left( 1+x \right) ^{\alpha}-1\sim \alpha x $

$(4). 1-\cos x\sim \frac{1}{2}x^2, x-\sin x\sim \frac{1}{6}x^3, \tan x-x\sim \frac{1}{3}x^3$

$(5). \mathrm{arc}\sin x-x\sim \frac{1}{6}x^3, x-\mathrm{arc}\tan \sim \frac{1}{3}x^3, \tan x-\sin x\sim \frac{1}{2}x^3$

$(6).$特别地, $1-\cos ^{\alpha}x=1-\left( 1+\left( \cos x-1 \right) \right) ^{\alpha}\sim -\alpha \left( \cos x-1 \right) \sim \frac{\alpha}{2}x^2$


$\mathrm{Eg}1.$ $(2024 \mathrm{I}). \underset{x\rightarrow 0}{\lim}\frac{\left( 1+ax^2 \right) ^{\sin x}-1}{x^3}=6 $ ,则$a=(~~~).$


解
$ 6$

$\mathrm{Eg}2.$ $(2023 \mathrm{I,II}).$ 当$x\rightarrow 0$时，函数$f\left( x \right) =ax+bx^2+\ln \left( 1+x \right) $ 与 $ g\left( x \right) =e^{x^2}-\cos x $是等价无穷小，则$ab=(~~~).$


解
$0$

$\mathrm{Eg}3.$ $(2020). \underset{x\rightarrow \frac{\pi}{2}}{\lim}\left( \sin x \right) ^{\frac{1}{1-\sin x}} $


解
$0$

$\mathrm{Eg}4.$ $\underset{x\rightarrow 0}{\lim}\frac{1-\cos ^{\alpha +\beta}x}{\sqrt{1-\cos ^{\alpha}x}\sqrt{1-\cos ^{\beta}x}}$


解
$\frac{\alpha +\beta}{\sqrt{\alpha \beta}}$

$\mathrm{Eg}5.$ $\underset{x\rightarrow \infty}{\lim}x\sin \frac{2x}{x^2+1}$


解
$2$

$\mathrm{Eg}6.$ $\underset{x\rightarrow 0}{\lim}\frac{x\ln \left( 1+x \right)}{1-\cos x}$


解
$2$

$\mathrm{Eg}7.$ $\underset{x\rightarrow 0}{\lim}\left( 1+3x \right) ^{\frac{2}{\sin x}}$


解
$e^6$

$\mathrm{Eg}8.$ $\underset{x\rightarrow 0^+}{\lim}\frac{1-\sqrt{\cos x}}{x\left( 1-\cos \sqrt{x} \right)}$


解
$1/2$

$\mathrm{Eg}9.$ $\underset{x\rightarrow +\infty}{\lim}\small{\ln \left( 1+2^x \right) \ln \left( 1+\frac{3}{x} \right)}$


解
$3\ln 2$

$\mathrm{Eg}10.$ $\underset{x\rightarrow 0^+}{\lim}\frac{\left( 1+x \right) ^{\frac{1}{x}}-e}{x}$


解
$-\frac{e}{2}$

$\mathrm{Eg}11.$ $\underset{x\rightarrow 0}{\lim}\left[ \frac{1}{\ln \left( 1+x \right)}-\frac{1}{x} \right] $


解
($\infty -\infty $型)

$\mathrm{Eg}12.$ $\underset{x\rightarrow +\infty}{\lim}\left[ x^2\left( e^{\frac{1}{x}}-1 \right) -x \right] $


解
($\infty -\infty $型)
$\frac{1}{2}$

$\mathrm{Eg}13.$ $(2020 \mathrm{I}). \underset{x\rightarrow 0}{\lim}\left[ \frac{1}{e^x-1}-\frac{1}{\ln \left( 1+x \right)} \right] $
