Logarithmic limit calculation

2024/10/24

Note

1.Logarithmic limit calculation.

$(1).$定式$1^{\infty}$型.
若$f\rightarrow 1,g\rightarrow \infty $,则$f^g\sim e^{g\left( f-1 \right)}$

$(2).$未定式$e^f-e^g$型.
若$f-g\rightarrow 0 $,则$e^f-e^g=e^g\left( e^{f-g}-1 \right) \sim e^g\left( f-g \right) $

文章详见《求极限：取对数的”套路”》.


$1. $定式$1^{\infty}$型.

$\mathrm{Eg}1.$ $\underset{x\rightarrow +\infty}{\lim}\left( \frac{a^{\frac{1}{x}}+b^{\frac{1}{x}}}{2} \right) ^x $


解
$\sqrt{ab}$

$\mathrm{Eg}2.$ $\underset{x\rightarrow +\infty}{\lim}\left( \frac{2}{\pi}\mathrm{arc}\tan x \right) ^x $


解
$e^{-\frac{2}{\pi}}$

$\mathrm{Eg}3.$ $\underset{x\rightarrow 0}{\lim}\left( \frac{\left( 1+x \right) ^{\frac{1}{x}}}{e} \right) ^{\frac{1}{x}} $


解
$e^{-\frac{1}{2}}$

$2.$ 未定式$1-1$型.

$\mathrm{Eg}1.$ $\underset{x\rightarrow 0}{\lim}\frac{1-\cos x\sqrt{\cos 2x}\sqrt[3]{\cos 3x}}{x^2} $


解
$3$

$\mathrm{Eg}2.$ $\underset{x\rightarrow 1}{\lim}\frac{\left( x^{3x-2}-x \right) \sin \left( 2\left( x-1 \right) \right)}{\left( x-1 \right) ^3} $


解
$6$

$3.$ 未定式$e^f-e^g$型.

$\mathrm{Eg}1.$ $\underset{x\rightarrow 0}{\lim}\frac{\left( 1+x \right) ^{\frac{1}{x}}-e}{x} $


解
$-\frac{e}{2}$

$\mathrm{Eg}2.$ $\underset{x\rightarrow 0}{\lim}\frac{e^{\left( 1+x \right) ^{\frac{1}{x}}}-\left( 1+x \right) ^{\frac{e}{x}}}{x^2} $


解
$\frac{e^{e+1}}{8}$

$\mathrm{Eg}3.$ $\underset{n\rightarrow \infty}{\lim}\left( \left( \left( n+1 \right) ! \right) ^{\frac{1}{n+1}}-\left( n! \right) ^{\frac{1}{n}} \right) $


解
$\frac{1}{e}$

$4.$ 未定式$1^{\infty} -1^{\infty}$型.

$\mathrm{Eg}1.$ $ \underset{x\rightarrow 0}{\lim}\frac{\left( 1+x \right) ^{\frac{1}{x}}-\left( 1+2x \right) ^{\frac{1}{2x}}}{x}$


解
$\frac{e}{2}$
